## Stack

- [TypeScript][] - Language
- [Vue3][] - Framework
  - [Vuex][] - State management
  - [Vue Router][] - Routing
- [Remult][] - Data access layer
- [Parcel][] - Build agent (transpilation, minification, bundling, etc.)
- [CSS Grid][] (mostly) - Layout paradigm
- [Less][] - CSS preprocessor (for media queries as variables)
